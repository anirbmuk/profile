## Quick Start & Documentation  
Local development server:  
```
yarn dev  
http://localhost:4200  
```
  
Local SSR run:  
```
yarn dev:ssr  
http://localhost:4300  
```
  
Production build and deployment:  
```
yarn deploy  
https://anirbmuk.appspot.com  
```
